# Git Workflows

A clean history helps code review and release tracing.

%%CW_CARD_START uid:CW-MOCK-002%%

### Rebase Checklist

1. Fetch the latest remote branches.
2. Rebase your feature branch onto the updated base branch.
3. Resolve conflicts carefully and continue the rebase.
4. Force-push with lease so you do not overwrite teammates unexpectedly.
